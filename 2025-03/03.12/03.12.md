# 📝 알고리즘 문제 풀이 - 2025-03-12

## 문제 📖

- **문제 링크:** [백준 - 스택(10828)](https://www.acmicpc.net/problem/10828)

- **문제 유형:** 구현, 자료 구조, 스택

- **난이도:** sliver4

- **풀이 시간:** ⏳ 10분

## ✍ 접근 방식

기본적인 스택 문제. 파이썬의 내장 함수를 이용해서 스택을 구현

1. 횟수를 입력 받음
2. 횟수 만큼 명령어 입력 받음
3. if 문으로 각 명령어가 들어오면 해야할 기능을 구현
